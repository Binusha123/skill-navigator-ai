# 🚀 Skillora AI

**Prove your skills. Plan your growth.**

Skillora AI is an AI-powered SaaS platform that goes beyond resumes to evaluate real skill proficiency. It analyzes a candidate’s resume and a job description, conducts a conversational assessment, identifies skill gaps, enhances the resume for better alignment, and generates a personalized learning roadmap to help users become job-ready faster.

---

## 🌟 Features

### 🧠 AI Skill Assessment

* Upload resume and paste job description
* AI generates role-specific questions
* Evaluates answers and assigns skill scores

---

### 📊 Skill Gap Analysis

* Compares required vs actual skills
* Identifies missing and weak areas
* Provides actionable insights

---

### 📚 Personalized Learning Plan

* Weekly roadmap (2–6 weeks)
* Focuses on **adjacent and realistic skills**
* Curated resources + time estimates

---

### 📝 Resume Enhancement (NEW ⭐)

* Aligns resume with job description
* Suggests:

  * Missing keywords
  * Improved bullet points
  * Better phrasing for impact
* Optimizes resume for **ATS + recruiters**

👉 Example:
❌ “Worked on ML project”
✅ “Developed a machine learning model achieving 85% accuracy using Python and Scikit-learn”

---

### 🎯 Job Readiness Score

* “You are 72% ready for this role”
* Based on skill match and assessment performance

---

### 📈 Skill Confidence Score

* Measures claimed vs demonstrated skills
* Displays confidence % per skill

---

## 🏗️ Tech Stack

* **Frontend:** React (Vite), Tailwind CSS
* **Routing:** React Router
* **Backend/DB:** Supabase (Auth + Database)
* **AI Integration:** OpenAI API (LLM)

---

## ⚙️ How It Works

1. Upload your resume
2. Paste a job description
3. Take AI-generated assessment
4. Get:

   * Skill scores
   * Gap analysis
   * Resume improvements
   * Job readiness score
   * Personalized learning plan

---

## 🗄️ Database Schema (Supabase)

* `profiles` → User data
* `resumes` → Parsed resume data
* `job_descriptions` → JD analysis
* `assessments` → Questions, answers, scores
* `learning_plans` → Roadmaps
* `confidence_scores` → Skill confidence
* `resume_suggestions` → Resume improvements
